NÚCLEO ENTERO

Ecos del Vacío — un reino sellado por matemáticas antiguas

OBJETIVO
Explora las 5 ramas del reino, resuelve operaciones con números enteros para ganar 3 habilidades y reunir 3 gemas. Con las 3 gemas, abre el portal final en el Vestíbulo.

CONTROLES
← → mover  ·  ↑ / Espacio saltar (doble salto si lo tienes)
Z / C / Ctrl golpear con el clavo  ·  Shift / X impulso (también golpea)

OPERACIONES CON ENTEROS
Cada trampa de pinchos en un pasillo tiene una operación (suma, resta, multiplicación, división o comparación). Resuélvela para retraer los pinchos — al resolver las 3 trampas de una rama, obtienes esa habilidad permanentemente.

CRIATURAS
Golpéalas con el clavo o el impulso para derrotarlas. Si te tocan sin que ataques, te devuelven a la entrada de la sala.

← → mover   |   ↑ / Espacio saltar (dos veces si tienes doble salto)   |   Z / C / Ctrl golpear con el clavo   |   Shift / X impulso (también golpea)   |   tocar una criatura sin golpear te devuelve a la entrada
